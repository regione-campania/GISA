Gestione Richieste Sviluppo Software

La funzionalità è accessibile dal cavaliere "Gestione Richieste" e consente di gestire le richieste delle attività di sviluppo.



Inserimento di un modulo



L'inserimento puo' avvenire tramite
  • Bottone giallo, "Aggiungi modulo A"
  • Collegamento "Aggiungi" sulla singola richiesta


Il bottone "Aggiungi modulo A" (a disposizione esclusivamente dei referenti regionali) è l'unico che consente di avviare una nuova richiesta.
In fase di aggiunta occorre indicare:
  • Descrizione della richiesta
  • Tags della richiesta (Insieme di termini riguardanti l'argomento trattato, per favorire la ricerca)
  • File
Il file da allegare deve rispettare la seguente sintassi:
Modulo [Lettera identificativa del modulo] [Progressivo della richiesta]

Es.

ModuloA045.pdf
Modulo_A 111.odt
Modulo A_006.doc
Modulo-A-900.pdf
Dal nome del file saranno estratti il tipo di modulo ed il progressivo della richiesta.

Una volta generata la richiesta, essa apparirà in una riga dedicata della tabella.
Da qui sarà possibile aggiungere i moduli B, C, CH, D tramite il link "Aggiungi" in corrispondenza della rispettiva colonna. In questo caso non sarà richiesta alcuna sintassi particolare nel nome del file.

Modifica Priorità

È possibile modificare la priorità di una richiesta, ma solo se non è stato ancora caricato un modulo C. (Questa funzionalità è disponibile solo per i referenti regionali.)

Note

Nel dettaglio di un singolo modulo sarà possibile aggiungere delle note. Le note saranno aggiunte in coda alle altre eventualmente presenti.

Filtro

È disponibile un filtro che permette di filtrare le richieste in base ai parametri inseriti.
Una volta riempiti i campi desiderati, fare click sul tasto 'CERCA' per eseguire la ricerca.
Se si desidera resettare la tabella e il filtro al loro stato iniziare cliccare invecec sul tasto 'RESET'.

Scadenzario

Tramite lo scadenzario è possibile consultare la lista delle richieste aperte e in attesa di risposta da più di due settimane.
